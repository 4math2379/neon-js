Components¶

These are methods used to convert JS objects into their respective neon-js implementation.

These methods are exposed for completeness but you are encouraged to use the constructors of the main objects Balance and Claims instead of manually recreating your own objects.

AssetBalance()¶
Arguments:
  • balance (Fixed8) – The total balance in this AssetBalance
  • unspent (Array.<Coin>) – Unspent coins
  • spent (Array.<Coin>) – Spent coins
  • unconfirmed (Array.<Coin>) – Unconfirmed coins
Coin()¶
Arguments:
  • index (number) – Index in list.
  • txid (string) – Transaction ID which produced this coin.
  • value (Fixed8) – Value of this coin.
ClaimItem()¶
Arguments:
  • claim (Fixed8) – Amt of gas claimable
  • txid (string) – Transaction hash of the originaating coin
  • index (number) – Index of coin in the output array
  • value (number) – Amount of NEO involved.
  • start (Fixed8) – Starting block. Optional.
  • end (Fixed8) – Ending block. Optional.
